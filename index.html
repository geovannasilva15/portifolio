Meus Projetos

🔐 SecurityCheck - Scanner de Segurança para Aplicações Web

O SecurityCheck é uma ferramenta desenvolvida em Python que analisa aplicações web em busca de vulnerabilidades comuns, como SQL Injection, Cross-Site Scripting (XSS) e cabeçalhos de segurança ausentes.

Funcionalidades:

  • ✅ Varredura de vulnerabilidades em URLs informadas pelo usuário.
  • ✅ Análise de cabeçalhos HTTP para verificar a implementação de segurança.
  • ✅ Detecção de falhas básicas em formulários e scripts maliciosos.
  • ✅ Geração de relatório detalhado com as vulnerabilidades encontradas.

Tecnologias Utilizadas: Python, Requests, BeautifulSoup, Flask, Linux.
